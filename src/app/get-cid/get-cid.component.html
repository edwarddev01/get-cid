Instrucciones paso a paso

Sigue estos pasos con cuidado para completar obteber tú ID de confirmación.

1

Paso 1: Coloca el token que te enviamos.

Con el token tienes acceso para poder obtener tú CID, recuerda que son de un solo uso.

2

Paso 2: Ingresa el ID de instalación

Ingresa el ID de instalación que te genero la licencia, sin espacio,s ni guiones.

3

Paso 3: Marca la casilla del reCAPTCHA.

Para verificar que eres humano.

4

Paso 4: Clic en "GET CID" espera por el ID de confirmación.

Espera mientras finaliza el proceso, que puede durar hasta 10 minutos no actualices, ni cierres la pagina.

5

Paso 5: Copia el ID de confirmación.

Una vez finalice, copia el CID y activa tu Office.
